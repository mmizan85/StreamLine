About StreamLine

StreamLine is a Chrome extension that applies real-time audio and video effects to media elements. It works on any website with media content, including nested frames and shadow DOMs.

All processing is done locally in your browser using the Web Audio API and CSS filters. No media data is sent to external servers.

This project is open source on GitHub. Contributions and feedback are welcome!
