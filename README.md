# 🚀 StreamLine: The Ultimate Browser Media Hub

**StreamLine** is a premium, high-performance Chrome Extension designed to unify and elevate your media consumption experience. Built with a sleek **Glassmorphism UI**, it allows you to manage every audio and video stream across your entire browser from a single, beautiful interface.

## ✨ Key Features

### 💎 Next-Gen User Interface

- **Glassmorphism Design:** A stunning "frosted glass" aesthetic with smooth CSS transitions and a dark mode optimized layout.
- **Dynamic Visualizer:** Real-time audio frequency animation that brings your music to life.
- **Smart Prioritization:** Automatically detects and promotes the "Currently Playing" media to the top for instant access.

### 🎵 Pro-Level Audio Tools

- **400% Volume Booster:** Safely amplify sound beyond standard limits for quiet videos or low-power speakers.
- **10-Band Manual Equalizer:** Build, save, and manage custom audio presets to suit your listening style.
- **Speech Enhancer:** Optimized frequency filtering to make dialogues clearer in movies and podcasts.

### 🎥 Advanced Video Control

- **One-Click PiP:** Pop any video into a floating window with a dedicated Picture-in-Picture button.
- **Speed Precision:** Control playback speeds from 0.5x to 16.0x without losing audio pitch.
- **Static Thumbnails:** High-performance metadata fetching for album art and video covers (No laggy frame-capturing).

### ⚡ Power User Automation

- **Multi-Tab Sync:** Automatically manages "Audio Focus" to prevent multiple videos from blaring at once.
- **Global Shortcuts:** Control media using custom keybindings (`Alt+Shift+P` for Play/Pause, etc.) even when the browser is minimized.

## 🚀 Installation Guide

StreamLine is developer-ready and easy to install locally:

1. **Download/Clone:** Download the repository as a ZIP or clone it via Git:
  
    ```bash
   git clone https://github.com/mmizan85/StreamLine.git
   ```

2. **Open Chrome Extensions:** Go to `chrome://extensions/` in your Chrome browser.
3. **Enable Developer Mode:** Toggle the "Developer mode" switch in the top right corner.
4. **Load Unpacked:** Click "Load unpacked" and select the `StreamLine` folder from your local machine.
5. **Pin to Toolbar:** Click the puzzle piece icon and pin StreamLine for easy access.
7. **Enjoy!** Click the StreamLine icon to open the popup and start controlling your media like a pro.

## 📖 How to Use

1. **Control:** Open the popup to see all active media tabs listed cleanly. Click the play/pause, skip, or PiP buttons to control each stream.
2. **Customize:** Click the Settings icon to access the Manual Equalizer and UI preferences.
3. **Shortcuts:** Use Alt+Shift+Up/Down to adjust volume and Alt+Shift+I for instant PiP.
4. **Clean UI:** If no background media is playing, the "Other Tabs" section automatically hides to save space.
5. **Audio Focus:** When you play a new video, StreamLine will automatically pause any other active media to prevent noise clashes.
6. **Visualize:** Enjoy the real-time audio visualizer that reacts to your music, making it a feast for the eyes as well as the ears.

## 🛠️ Tech Stack

- ***Architecture:*** Manifest V3 (Service Workers & Declarative Net Request)

- ***Engine:*** Web Audio API (BiquadFilterNodes for EQ & GainNodes for Volume)

- ***UI:*** Glass Pure HTML5/CSS3 (Flexbox/Grid/Backdrop-Filters)

- ***Logic:*** Vanilla JavaScript (ES6+ Modules)
